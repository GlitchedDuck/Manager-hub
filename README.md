# Manager Hub & TAG Training

Complete platform for check-ins, actions, and training management.

## Quick Start
```bash
pip install -r requirements.txt
python generate_enhanced_data.py
streamlit run app_enhanced.py
```

## Features

- ✅ Check-in notes between 1-2-1s
- ✅ Action tracking
- ✅ Training Matrix with tickable progress
- ✅ Sytner Training bookings
- ✅ Learning Resources tracking
- ✅ Expense integration
